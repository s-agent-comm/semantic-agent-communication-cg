# W3C Semantic Agent Communication CG - Specifications

This directory contains the official Community Group Reports (CGRs) for the Semantic Agent Communication standard.

## Overview Documents

*   **[Explainer](./explainer.md)**: Start here! A concise overview of the problem, solution, and user needs.
*   **[Primer](./primer.md)**: A deep dive into the vision, philosophy, and high-level architecture.
*   **[Architecture & Requirements](./architecture.md)**: Detailed use cases, requirements, and the "Unified Narrative" architectural model.

## Normative Specifications

These documents define the standard. Implementations must conform to them.

*   **[Core Ontology](./core-ontology.md)**: The RDF/OWL vocabulary for Agents, Intents, Delegations, and Narratives.
*   **[AgentIDL](./agent-idl.md)**: The Interface Description Language for defining agent capabilities.
*   **[Security & Attestation](./security.md)**: The security model, including DIDs, Verifiable Credentials, and Hardware Attestation.
*   **[Protocol & Transport](./protocol.md)**: Serialization formats (JSON-LD) and transport bindings.

## Other Resources

*   `vocabulary/`: Source files for the ontology (Turtle/TTL).
*   `w3c/`: W3C-specific templates and resources.
