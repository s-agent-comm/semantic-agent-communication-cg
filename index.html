Semantic Agent Communication Community Group

Official website of the W3C Semantic Agent Communication Community Group .

The group develops semantic foundations, ontologies, and interoperability models for autonomous software agents. Work includes the definition of interaction semantics, identity and verification primitives, narrative and ledger models, and the AgentIDL interface description language.
